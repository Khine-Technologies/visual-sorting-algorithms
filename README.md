# Visual-Sorting-Algorithms
Sorting Algorithms with CLI sorting output.

## Supported Algorithms
- [x] Bubble Sort
- [ ] Selection Sort
- [ ] Insertion Sort
- [ ] Merge Sort
- [ ] Quick Sort
- [ ] Heap Sort
- [ ] Iterative Heap Sort
- [ ] Counting Sort
- [ ] Radix Sort
- [ ] Bucket Sort

## Installation MacOS
Make a new Virtual Environment
```commandline
python3 -m venv venv
```
```commandline
source ./venv/bin/activate
```
```commandline
pip install -r requirements.txt
```
